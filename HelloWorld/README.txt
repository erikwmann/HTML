- All HTML documents must start with a type decleration. ex <!DOCTYPE html>
- The HTML document itself begins and ends with the <html> and </html> tags
- The visible part of the HTML is between the <body> and </body> tags

- HTML element consist of start and end tags. The content is in between.
- HTML elements can be nested. All HTML documents are composed of nested elements.
- Always add the end tags even if optional.
- HTML is not case sensitive

- All HTML elements can have attributes
- Attributes are name/value pairs. ex name="value"
